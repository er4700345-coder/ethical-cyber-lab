## 💻 About This Project

Ethical Cyber Lab is a **safe and fully legal web penetration testing toolkit**.  
It is designed for **learning, research, and authorized penetration testing only**.  

- Test for SQL Injection (SQLi), XSS, CSRF  
- Automated reconnaissance & info gathering  
- Modular Python & Bash scripts  
- Organized structure for professional coding practice  

> **Ethical Reminder:** Only test on systems you own or have permission to test.

---

## 🛠 Features

- 🔍 SQL Injection detection  
- 🌐 Cross-Site Scripting (XSS) checks  
- 🛡 CSRF vulnerability detection  
- 📡 Automated reconnaissance scripts  
- 💻 Python & Bash modular scripts
